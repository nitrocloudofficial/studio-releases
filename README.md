# NitroStudio

NitroStudio is a desktop application for building and managing AI-powered applications with a visual interface.

## Download

Download the latest release for your platform:

| Platform | Download |
|----------|----------|
| Windows | [NitroStudio-latest.exe](https://github.com/nitrocloudofficial/studio-releases/releases/latest) |
| macOS (Apple Silicon) | [NitroStudio-latest.dmg](https://github.com/nitrocloudofficial/studio-releases/releases/latest) |
| macOS (Intel) | [NitroStudio-latest.dmg](https://github.com/nitrocloudofficial/studio-releases/releases/latest) |
| Linux | [NitroStudio-latest.AppImage](https://github.com/nitrocloudofficial/studio-releases/releases/latest) |

## Installation

### Windows
1. Download the `.exe` installer
2. Run the installer and follow the prompts
3. Launch NitroStudio from your Start Menu

### macOS
1. Download the `.dmg` file
2. Open the DMG and drag NitroStudio to your Applications folder
3. Launch NitroStudio from Applications
4. If you see a security warning, right-click the app and select "Open"

### Linux
1. Download the `.AppImage` file
2. Make it executable: `chmod +x NitroStudio-*.AppImage`
3. Run the AppImage: `./NitroStudio-*.AppImage`

## System Requirements

### Windows
- Windows 10 or later (64-bit)
- 4 GB RAM minimum (8 GB recommended)
- 500 MB free disk space

### macOS
- macOS 10.15 (Catalina) or later
- 4 GB RAM minimum (8 GB recommended)
- 500 MB free disk space

### Linux
- Ubuntu 18.04+ or equivalent distribution
- 4 GB RAM minimum (8 GB recommended)
- 500 MB free disk space

## Features

- **Visual Workflow Builder**: Create AI workflows with a drag-and-drop interface
- **Multi-Model Support**: Connect to various AI providers and models
- **Project Management**: Organize your AI projects in one place
- **Real-time Preview**: Test and iterate on your workflows instantly
- **Export & Deploy**: Export your workflows for production use

## Auto-Update

NitroStudio automatically checks for updates and will notify you when a new version is available. You can also manually check for updates from the app menu.
